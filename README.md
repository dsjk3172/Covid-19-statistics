# 소개
2019년 시작된 Covid-19는 현재 2021년이 되어서도 진정되지 않고 여전히 사람들은 불안에 떨고 있습니다.
Covid-19에 관련된 웹사이트들은 많지만 그렇기에 반대로 필요한 정보들에 대한 접근성이 떨어지는 상황입니다

그렇기에 정보들을 한 곳에 모아 정리하여 이용자의 수고를 덜어주고, 편의를 도모하고자 웹사이트를 만들고자 하였습니다.
